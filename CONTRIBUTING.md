# Contributing to OmniRent

Thank you for your interest in contributing.

## Workflow
1. Fork the repository
2. Create a feature branch
3. Make your changes
4. Run tests locally
5. Submit a pull request

## Requirements
- Follow existing code structure
- Maintain test coverage (Jest + Supertest)
- Ensure no breaking changes to escrow logic

## Focus Areas
- Security improvements
- UI/UX enhancements
- Performance optimizations
